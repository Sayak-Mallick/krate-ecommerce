# Krate E-Commerce
A high-performance, minimalist full-stack e-commerce solution with a dedicated administrative dashboard.

### Overview
Krate is a "headless-first" e-commerce platform designed for speed and scalability. It features a sleek, responsive customer storefront and a robust administrative panel for inventory, order, and user management.

Key Features
🛒 Storefront: Dynamic product filtering, persistent cart via Redux/Local Storage, and secure checkout.

🛡️ Admin Panel: Full CRUD operations for products, real-time order status updates, and user role management.

🔐 Security: JWT-based authentication, password hashing with Bcrypt, and protected API routes.

📊 Data Management: Relational-style modelling in MongoDB for orders and category associations.

📱 Responsive: Mobile-first design using Tailwind CSS / Material UI.

### Tech Stack

Layer, Technology
Frontend, "React.js, Redux Toolkit, Tailwind CSS"
Backend, "Node.js, Express.js"
Database, MongoDB (Mongoose ODM)
Auth, JSON Web Tokens (JWT)
Payments, Stripe API (Optional/Planned)
